# FOCUSENSE PRO – Project Showcase Website

This is a modern, aesthetic, and interactive multi-page website created to present the FOCUSENSE PRO project in a visually engaging way.

The website explains the complete working of the system including its features, architecture, technologies, and results using a clean and futuristic UI design.

- Cyan & deep blue gradient theme  
- Glassmorphism UI (frosted glass cards)  
- Smooth animations and transitions  
- Floating particle background effects  
- Responsive design (mobile + desktop)  
- Modern and minimal layout  

- Home  
- About Project  
- Features  
- How It Works  
- Technologies  
- Results & Output  
- Contact / Developer Info  

The purpose of this website is to provide a clear and visually appealing explanation of the project, making it easy for users, evaluators, and recruiters to understand the system.
